# 转二进制

十进制转二进制：整数部分

```python
def binary_int(num):
    res = ''
    while num > 1:
    	mod = num % 2
    	res = str(mod) + res
    	num = num // 2
    return '1' + res
```

十进制转二进制：小数部分

输入是一个去掉整数部分的小数，也就是`0.xxxx`

```python
def binary_float(fl):
    res = ''
    while fl > 0.0000001: # 防止无限循环，可根据需求调整
        fl *= 2
        res += str(fl)[0]
        if fl >= 1:
            fl -= 1
    return '0.' + res
```



# 判断一个数是否是2的次方

`&`是位运算(Python, Java都是)，判断的是数字的二进制的每一位的是否相等，相等则1，不等则0.

例如：

- 2^4^ = 16
- 16二进制是 10000
- 15二进制是01111
- 因为他们的二进制每一位都不相同，16&15就是00000
- 所有2^n^的二进制都是1后面跟n个0，而2^n^-1的二进制都是n个1，所以他们一定遵循&后等于0

```python
def power_of_two(n):
    if n & (n-1) == 0:
        return True
    else:
        return False
```
