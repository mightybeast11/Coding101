# 贪心

Time: O(nlogn), because of sort, 贪心只是O(n)

Space: O(1), count, i, j

```python
class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        count = 0
        g.sort()
        s.sort()
        i, j = 0, 0

        while i < len(s) and j < len(g):
            if s[i] >= g[j]:
                count += 1
                i += 1
                j += 1
            else:
                i += 1
        return count
```

总是先用尽可能小的饼干，满足胃口最小的孩子。
